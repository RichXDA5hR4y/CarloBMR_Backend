# ⚙️ CARLO BMR SHOP - Backend (Laravel/PHP API)

Ini adalah repositori Backend (API) untuk **CARLO BMR SHOP**, yang menangani manajemen data produk, pengguna, dan logika keranjang belanja.

## 🚀 Fitur Utama
* **RESTful API**: Menyediakan endpoint untuk CRUD produk dan user.
* **Autentikasi Sanctum**: Sistem login aman menggunakan token.
* **Manajemen Database**: Integrasi dengan MySQL untuk penyimpanan data persisten.

## 🛠️ Teknologi yang Digunakan
* **PHP & Laravel Framework**: Framework utama untuk logika backend.
* **MySQL (Aiven Cloud)**: Database utama yang di-host secara cloud.
* **Composer**: Untuk manajemen dependensi PHP (folder vendor tidak di-upload).

## 📦 Cara Menjalankan Secara Lokal
1. Clone repositori ini:
   ```bash
   git clone [https://github.com/RichXDA5hR4y/CarloBMR_Backend.git](https://github.com/RichXDA5hR4y/CarloBMR_Backend.git)

2. Instal dependensi:
   ```bash
   composer install
   
3. Konfigurasi file .env dengan kredensial database Anda.
4. Jalankan migrasi dan server:
   ```bash
   php artisan migrate
   php artisan serve

🌐 Deployment
Walau ada error diawal, tapi untuk pengembangan/perbaikan lebih lanjut aplikasi ini dipersiapkan untuk dideploy menggunakan Render atau apapun tools sejenis berbeda yang bisa dipakai selanjutnya
